# Day 4 - PUT and DELETE

Tool: Postman

PUT Request:
Used to update existing data.

Endpoint:
https://jsonplaceholder.typicode.com/users/1

Status Code:
200 OK

DELETE Request:
Used to remove data from server.

Endpoint:
https://jsonplaceholder.typicode.com/users/1

Status Code:
200 OK

Learning:
PUT updates data, DELETE removes data from server.
